1. Назначение

Данный фреймворк предназначен для создания многослойных нейронных сетей и состоит из:

- функций загрузки датасетов - 2 шт;

- функций работы с данными - 5 шт;

- функций активации в паре с функциями производных от функций активации - 3 шт;

- функций потерь в паре с функциями производных от функций потерь - 2 шт;

- функций оптимизаторов - 3 шт;

- описания класса Layer (слой);

- описания класса NeuralNetwork (нейронная сеть).

2. Функции загрузки датасетов

2.1 iris_loader()

Возвращает:

X — массив признаков формы (150, 4), где каждая строка — это один образец, а столбцы представляют длину и ширину чашелистика и лепестка;

y — массив меток классов формы (150,), содержащий значения {0, 1, 2}, которые представляют три вида ирисов:

0 — Iris Setosa;

1 — Iris Versicolor;

2 — Iris Virginica.

2.2 mnist_loader(sample_size = 10000)

Принимает на вход параметр sample_size, указывающий, сколько образцов данных мы хотим взять из настоящего датасета.
Возвращает:

X — массив изображений размером (sample_size, 784), где каждое изображение 28×28 пикселей развернуто в одномерный вектор из 784 значений, где значения нормализованы в диапазоне [0;1] делением на 255;

y — массив меток классов размером (sample_size,), содержащий цифры от 0 до 9.

3. Функции работы с данными

3.1 one_hot_encode()

Принимает на вход: массив меток классов y.
Возвращает (пример):
[[0. 0. 1.] # 'cat' → [0 0 1]
[0. 1. 0.] # 'dog' → [0 1 0]
[1. 0. 0.] # 'bird' → [1 0 0]
[0. 1. 0.] # 'dog' → [0 1 0]
[0. 0. 1.]] # 'cat' → [0 0 1]

3.2 map(X,func)

Принимает на вход:

X — массив (или список) значений;

func — функцию, которую нужно применить к каждому элементу X.

Возвращает: numpy-массив, где к каждому элементу X применена переданная функция func.

3.3 shuffle(X,y)

Принимает на вход:

X — массив (или список) признаков;

y — массив (или список) меток. 

Возвращает: X и y в перемешанном порядке, но с сохранением соответствия между X и y в каждой единицы данных.

3.4 batch(X,y,batch_size)

Принимает на вход:

X — массив (или список) признаков;

y — массив (или список) меток.

batch_size — размер пакета (батча), то есть количество примеров в одном батче.

Возвращает:

batches_X — список, содержащий батчи данных X, где каждый батч — это подмассив из batch_size элементов (если количество элементов не делится нацело на batch_size, последний батч может быть меньше остальных);

batches_y — список, содержащий батчи меток y, где каждый батч — это подмассив из batch_size элементов (если количество элементов не делится нацело на batch_size, последний батч может быть меньше остальных);

3.5 train_test_split(X, y, test_size=0.2, random_state=None)

Принимает на вход:

X — массив (или список) признаков (входных данных).

y — массив (или список) меток (целевых значений).

test_size (по умолчанию 0.2) — доля данных, выделяемая в тестовую выборку (может быть числом от 0 до 1 или целым количеством примеров).

random_state (по умолчанию None) — если задано число, фиксирует случайное перемешивание (для воспроизводимости результатов).

Возвращает:

X_train — массив (список) обучающих данных.

X_test — массив (список) тестовых данных.

y_train — массив (список) обучающих меток.

y_test — массив (список) тестовых меток.

6. Функций оптимизаторов

6.1 sgd(weights, gradient, learning_rate) 

Принимает на вход:

weights — массив текущих весов (numpy.ndarray), размерность (n_features, n_outputs);

gradient — массив градиентов (частных производных функции потерь по весам) (numpy.ndarray), той же размерности, что и weights;

learning_rate — коэффициент скорости обучения (float).

Возвращает: обновленные веса (numpy.ndarray) после применения градиентного спуска.

6.2 momentum_sgd(weights, gradient, learning_rate, velocity, momentum=0.9) 

Принимает на вход:

weights — массив текущих весов (numpy.ndarray), размерность (n_features, n_outputs);

gradient — массив градиентов (частных производных функции потерь по весам) (numpy.ndarray), той же размерности, что и weights;

learning_rate — коэффициент скорости обучения (float);

velocity — массив скоростей обновления весов (numpy.ndarray), той же размерности, что и weights;

momentum — коэффициент момента (float, по умолчанию 0.9).

Возвращает: 

Обновленные веса (numpy.ndarray) после применения градиентного спуска;

Обновленное значение скорости (numpy.ndarray).

6.3 sign_sgd(weights, gradient, learning_rate)

7. Класс Layer (слой) Описывает один слой нейронной сети: инициализацию и методы класса forward и backward.

7.1 init(self, input_size, output_size, activation_func, activation_derivative, optimizer)

Принимает на вход:

input_size (int) — количество входных нейронов (размер входного слоя);

output_size (int) — количество выходных нейронов (размер текущего слоя);

activation_func (function) — функция активации слоя (например, sigmoid, relu, linear);

activation_derivative (function) — производная функции активации, используемая при обратном распространении ошибки;

optimizer (function) — функция оптимизации (например, sgd, momentum_sgd).

7.2 forward(self, X) - метод прямого прохода по слою

Принимает на вход:

X (numpy.ndarray) — входные данные.

Возвращает: self.output (numpy.ndarray) — выходное значение слоя после применения функции активации.

7.3 backward(self, d_output, learning_rate, momentum) - метод обратного прохождения, реализующая обновление весов

Принимает на вход:

d_output (numpy.ndarray) — градиент ошибки, поступающий от следующего слоя;

learning_rate (float) — коэффициент скорости обучения;

momentum (float) — коэффициент импульса (используется в оптимизаторе momentum_sgd).

Возвращает: d_input (numpy.ndarray) — градиент ошибки по входным данным (X) для передачи на предыдущий слой.

8. Класс NeuralNetwork (нейронная сеть) описывает класс нейронной сети

8.1 init(self, activation_func=sigmoid, activation_derivative=sigmoid_der, optimizer=sgd, loss_function='mse')

Принимает на вход:

activation_func (функция, по умолчанию sigmoid);

activation_derivative (функция, по умолчанию sigmoid_der);

optimizer (функция, по умолчанию sgd);

loss_function (str, по умолчанию 'mse').

8.2 add_layer(self, input_size, output_size, activation_func=None, activation_derivative=None)

Принимает на вход:

input_size (int) - количество входных нейронов в слое (размерность входных данных для данного слоя);

output_size (int) - количество выходных нейронов в слое (размерность выходных данных этого слоя);

activation_func (функция, по умолчанию None);

activation_derivative (функция, по умолчанию None).

8.3 forward(self, X)

Применяет функцию прямого прохождения Layer.forward() ко всем слоям сети.

Принимает на вход: массив признаков Х.
Возвращает: обновлённый результат Х.

8.4 backward(self, d_output, learning_rate, momentum)

Применяет функцию обратного прохождения класса Layer.backward() ко всем слоям в сети.

Принимает на вход:

d_output (ndarray) - градиент ошибки, поступающий от следующего слоя.

learning_rate (float) - шаг обучения;

momentum (float) - моментум, используется в том случае, если оптимизатор - это momentum_sgd.

8.5 train(self, X, y, epochs=100, learning_rate=0.01, batch_size=32, momentum=0.9)

Принимает на вход:

X - матрица признаков;

y - матрица целевых значений;

epochs - число циклов обучения;

learning_rate - шаг обучения;

batch_size - размерность данных;

momentum - моментум, параметр, использующийся в тех случаях, когда оптимизатор слоя - это momentum_sgd.
